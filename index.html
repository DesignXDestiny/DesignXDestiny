FrontEnd Development

Front End Development is like an evolved form of poetry or English. It’s like English class in the way that you develop your ideas like an essay, and that’s literally mostly written in English. It is like poetry because less is more in the same way. Much of the drafting process is me trying to say the same thing with less code or words. Developing web pages, or anything this way is when I’m trying to create something to share with someone. This is when I create with my audience in mind.
